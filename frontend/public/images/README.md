# Image Slider Setup Instructions

## Required Images
Place these images in the `/public/images/` folder:

1. **slide1.jpg** - Your first uploaded image (University Campus)
2. **slide2.jpg** - Your second uploaded image (Student Learning) 
3. **slide3.jpg** - Optional third image (University Building)

## Image Specifications
- Recommended size: 1920x1080 pixels (16:9 ratio)
- Format: JPG or PNG
- File size: Under 2MB for optimal loading

## Current Slider Configuration
- Auto-play: 5 seconds per slide
- Manual navigation: Previous/Next buttons
- Dot indicators for direct navigation
- Responsive design for all screen sizes

## Features
- Smooth fade transitions
- Overlay text with call-to-action buttons
- Mobile-friendly touch controls
- Keyboard navigation support

## Troubleshooting
If images don't appear:
1. Check file names match exactly
2. Verify images are in `/public/images/` folder
3. Ensure proper image format (JPG/PNG)
4. Check browser console for errors
